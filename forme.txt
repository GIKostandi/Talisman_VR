Запушить:
git add .
git commit -m "какие изменения"
git push -u origin main

Обновить у себя (Все обновления обязательно в папке Talisman_vr):
git pull origin main

скачать репу с гита
git clone https://github.com/GIKostandi/Talisman_VR.git

Удалить внесенные собой изменения локально:
git reset --hard origin/main
